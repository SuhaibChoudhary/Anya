Anya Bot Privacy Policy

This privacy policy explains how Anya Bot ("the bot") collects and uses data within Discord servers.

1. What Data We Collect

  • Server IDs, Channel IDs, and User IDs for internal functionality.
  • Message content (only when the Message Content Intent is enabled) to enable DJ functionality and prefixless commands in specific channels.
  • Bot usage statistics like command usage and error logs.

2. Why We Collect It

We use this data to:

  • Allow DJ Role to execute commands without a prefix in designated channels.
  • Track command usage to improve bot performance.
  • Prevent abuse through features like global blacklisting.

3. Data Sharing

We do not sell, share, or trade your data with any third party. All data is kept secure and used solely for bot functionality.

4. Opting Out

If you do not want the bot to track message content, you can disable DJ Mode in your server or remove the bot.

5. Contact

For any concerns or inquiries, contact the developer via Discord or email: yourmail@example.com
